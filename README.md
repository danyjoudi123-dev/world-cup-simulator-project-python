# world-cup-simulator-project-python
My first ever python project from scratch that simulates a football World Cup. It is currently not finished but functional to simulate one group.

## Features
- Simulates group stage matches
- Generates goals based on team ratings
- Builds and sorts a league table
- Sorts by points, goal difference and goals scored

## Future Improvements
- Support all 48 teams
- Multiple groups
- Knockout brackets
- Improved match engine
- Improved goal generation based on more accurate rankings
